Do / Don't

Do

Use one icon set consistently. Keep stroke width uniform (2px default). Pair icons with text labels for ambiguous actions.

Don't

Mix icon sets (Lucide + Material + emoji). Scale icons non-proportionally. Rely on color alone to communicate meaning.

Do / Don't

Do

Animate transform and opacity only (GPU-accelerated). Respect prefers-reduced-motion. Keep durations ≤ 300ms for UI feedback.

Don't

Animate layout properties (width, height, top). Chain long sequences for utilitarian actions. Autoplay background animation in content. 

@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
